# Images de Fruits à Ajouter

Pour améliorer le site d'export de fruits, voici les images recommandées à placer dans `/public/home/` :

## Images Hero / Bannière
1. **hero.jpg** - Image de fond avec des fruits frais tropicaux variés
   - Suggestion : Assortiment de mangues, ananas, bananes sur fond naturel
   - Dimension recommandée : 1920x1080px

2. **full-kids.jpg** → Renommer en **plantation.jpg**
   - Image de vergers ou plantation de fruits
   - Dimension recommandée : 1920x900px

## Note Importante

Toutes les images doivent être :
- **Haute résolution** (min 1200px de largeur)
- **Format JPG** optimisé pour le web
- **Licence commerciale** autorisée
- **Aspect professionnel** et appétissant
- **Éclairage naturel** de préférence
